Welcome to Track Print

This project demonstrates browser fingerprinting — a technique websites use to collect unique data from your browser and device to potentially track or identify you.

Below are some available commands you can try:

--- AVAILABLE COMMANDS ---

help              : Display this help message and list all available commands
print-fingerprint : Show browser and device data that may uniquely identify you
show-location     : Attempt to retrieve your current geolocation
track             : Emulate how trackers combine data to fingerprint you
clear             : Clear the terminal screen
about             : Learn more about fingerprinting and online privacy

Start typing a command to explore how your digital fingerprint is formed. This tool is built to raise awareness about data transparency and online tracking practices.
